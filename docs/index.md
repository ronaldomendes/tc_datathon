# Datathon Passos Mágicos

Projeto de **previsão do risco de defasagem escolar** com base nos dados da pesquisa PEDE 
(Passos Mágicos). Inclui pipeline de ML (XGBoost), API FastAPI, interface Gradio e integração 
com MLflow.

Use o menu ao lado para acessar a visão geral, estrutura do projeto, como executar, 
documentação da API e etapas do pipeline de Machine Learning.
